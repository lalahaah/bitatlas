# 보안 정책 (Security Policy)

## 지원하는 버전
현재 보안 업데이트가 제공되는 버전은 다음과 같습니다.

| 버전 | 지원 여부 |
| :--- | :---: |
| 최신 버전 (Main) | ✅ |
| 이전 버전 | ❌ |

## 취약점 보고 방법
보안 취약점을 발견하셨다면, 공개적인 이슈(Issue)를 올리는 대신 아래 이메일로 비공개 보고를 부탁드립니다.

* **이메일:** **hello@nextidealab.app**

보고해 주실 때 취약점의 상세 내용과 재현 방법을 포함해 주시면 확인 후 최대한 빠르게 답변드리겠습니다. 프로젝트의 안전을 위해 도움을 주셔서 감사합니다.
